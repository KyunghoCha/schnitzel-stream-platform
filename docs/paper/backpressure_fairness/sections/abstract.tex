\begin{abstract}
This paper studies fairness-sensitive failure behavior in stream processing under backpressure.
We evaluate four queue-overflow policies across four reproducible workloads and report latency,
reliability, group-gap metrics, and harm-weighted cost.
The native runtime analysis is the core claim, while ROS2 comparison is reported as appendix evidence.
All experiments, statistics, and tables are generated from versioned configs and scripts.
\end{abstract}
