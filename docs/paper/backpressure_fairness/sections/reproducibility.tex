\section{Reproducibility Package}
The submission package includes:
\begin{itemize}
  \item versioned matrix and analysis specs,
  \item generated artifact manifest,
  \item command log for re-execution,
  \item checklist for quality gates and output contracts.
\end{itemize}

All commands are expected to run in the \texttt{schnitzel-stream} conda environment with \texttt{PYTHONPATH=src}.
