\section{Reproducibility Package}
Single source of truth for this manuscript:
\texttt{outputs/experiments/backpressure\_fairness/paper\_submission\_final\_ros2}.

The package includes:
\begin{itemize}
  \item frozen configs: \texttt{final\_matrix\_v1.yaml}, \texttt{analysis\_spec\_v1.yaml},
  \item aggregate outputs: \texttt{aggregate/backpressure\_aggregate.json} and CSV companions,
  \item ROS2 comparison outputs: \texttt{ros2\_comparison.json/csv}, \texttt{ros2\_reproducibility.csv},
  \item manuscript tables: \texttt{package/latex\_tables/*.tex},
  \item execution and QA records: \texttt{package/artifacts\_manifest.json},
  \texttt{package/repro\_commands.md}, \texttt{package/submission\_checklist.md}.
\end{itemize}

Interpreter split is required:
\begin{itemize}
  \item native stages: conda \texttt{schnitzel-stream} Python with \texttt{PYTHONPATH=src},
  \item ROS2 stages: \texttt{/usr/bin/python3} after sourcing \texttt{/opt/ros/humble/setup.bash}.
\end{itemize}

The final manifest for this artifact set reports:
\texttt{ros2\_baseline\_stage\_status=ok},
\texttt{ros2\_compare\_stage\_status=ok},
and all configured quality gates as passed.
