# Ingest Package

## English

Reserved for future ingest expansion (producer/consumer, buffering, rate control).
Currently, frame ingest is handled by `src/ai/pipeline/sources/` (package) and sampling
by `src/ai/pipeline/sampler.py`.

## 한국어

향후 ingest 확장(프로듀서/컨슈머, 버퍼링, 레이트 제어)을 위한 영역이다.
현재 프레임 입력은 `src/ai/pipeline/sources/`(패키지), 샘플링은
`src/ai/pipeline/sampler.py`가 담당한다.
